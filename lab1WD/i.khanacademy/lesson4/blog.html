Entry 1: My First Steps in Programming

Starting my programming journey was both exciting and overwhelming. I began with Python, a language recommended for beginners due to its simplicity and readability. Writing my first "Hello, World!" program made me feel like a real programmer, even though it was just the beginning.

Over time, I learned basic concepts like variables, loops, and functions. The more I practiced, the more confident I became in solving small problems. This experience sparked my curiosity to explore more about coding.

Entry 2: Learning HTML and CSS

After understanding the basics of programming, I decided to dive into web development. Learning HTML and CSS was a game-changer. HTML allowed me to structure content, while CSS made it visually appealing.

I practiced by creating simple web pages, like a personal portfolio and a contact form. Customizing the styles using CSS properties like color, font-family, and margin was both challenging and rewarding.
